# Investor Portal Media Assets

Add the following images to customize the investor presentation:

| File | Description |
|------|-------------|
| `hero.jpg` | Hero section – plot aerial view or villa render |
| `plot.jpg` | The Plot section – plot plan or topographic survey |
| `concept.jpg` | Development Concept section – architectural render |
| `location.jpg` | Location section – map or area overview (optional) |

Recommended dimensions: 1920×1080 or 16:9 aspect ratio for hero; 4:3 for plot/concept.
